15강. 포인터 함수에서의 포인터 매개변수

강의 내용
 - 포인터에 대한 기초 내용
 - 값을 가리킨다(pointing)라는 의미에서 포인터 라고 이름이 붙었으며 * 기호로 표기된다.
 - 특정 변수를 가리키는 값으로 메모리의 주소값을 의미한다.
 - 포인터의 사용은 변수 선언 시, 포인터 변수 사용 시 변수명 앞에 *을 붙여 사용한다.
 - 포인터에 직접 값을 대입하기 위해서는 new 키워드를 통해서 공간을 할당 해주어야한다.
   ex) int *pPointer = new int;
        *pPointer = 10;
 - new 키워드를 사용하여 공간을 할당받은 경우 꼭 사용이 끝나면 delete키워드를 사용하여
   할당 해제를 해주어야한다. 하지 않으면 메모리 누수가 발생한다.
   ex ) delete pPointer;
 - 포인터로 값을 전달 시 값이 들어 있는 공간에 대한 정보를 전달하는 것이기 때문에 값에 대한
   직접적인 접근이 가능하다.
