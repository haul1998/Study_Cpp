   이번 강의 영상은 visual studio 2022로 진행했는데 return 0;를 사용하면 오류가 뜹니다.
   기존에 사용하지 않았던 system("pause")를 사용하는데 강의에는 나오지 않았지만 혼자 코딩해본 결과
   코드 진행을 한번 멈추는 코드인걸 알았는데 왜 return 0;를 더이상 쓰지 않고 system("pause")를 쓰나요?
